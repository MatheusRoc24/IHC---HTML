CSS Grid Layout

CSS Grid Layout é um sistema de layout bidimensional do CSS criado para organizar elementos em linhas e colunas de forma estruturada e responsiva. Ele facilita a construção de interfaces modernas sem depender de float ou frameworks externos.

Os principais conceitos são:

  • Container Grid: elemento pai que recebe display: grid.
  • Grid Items: elementos filhos organizados dentro da grade.
  • Colunas e linhas: definidas com grid-template-columns e grid-template-rows.
  • Espaçamento: controlado com gap.
  • Posicionamento: itens podem ocupar múltiplas linhas ou colunas usando grid-column e grid-row.
  • Responsividade: funções como repeat(), auto-fit, auto-fill e minmax() permitem layouts adaptáveis.

Exemplo básico

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    gap: 10px;
}

Nesse exemplo:

  • display: grid ativa o Grid.
  • 1fr divide o espaço igualmente (1 fração do espaço disponível).
  • gap adiciona espaçamento entre os elementos.

Vantagens

  • Organização mais profissional do layout.
  • Melhor controle de alinhamento bidimensional.
  • Facilidade para criar dashboards, galerias e páginas responsivas.
  • Reduz a necessidade de media queries complexas.

O CSS Grid é indicado principalmente para estruturar páginas completas e layouts complexos.
